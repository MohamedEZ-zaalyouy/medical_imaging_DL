# medical_imaging_DL

Dataset The dataset contains +30k OCT scans of the retina, the scans belongs to one of the following categories:  

CNV: Choroidal neovascularization اتساع الأوعية الدموية المشيمية
DRUSEN : first signs of Age Macular Degeneration أولى علامات التنكس البقعي للعمر 
DME : Diabetic Macular Ederma درما البقعي السكري 
NORMAL : Healthy scans فحوصات صحية
